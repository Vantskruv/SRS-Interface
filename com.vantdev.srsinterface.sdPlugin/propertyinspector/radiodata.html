Frequency change
To increase the frequency, set a value of i.e. 0.025
To decrease, assign a negative value.
I.e. to decrease a current frequency of 162.500 to 162.250, set the field to:
-0.25

Radio appearance
How to ... Customize how your button should appear. Everything inside the brackets [ ] should contain a valid tag which is described below. Everything else is treated as static text. If you want to show brackets [ ] in the button, prepend them with a \ i.e:
\[ Static text including brackets \]
A new line is treated as a new line in the button.
List of tags:
[RadioName] This shows the name of the radio
[Clients] This shows the number of clients
[SenderName] On radio transmission, this shows the sender name.
[Freq] Shows the radio frequency nnn.nnn in Mhz
[FreqMhz] As above, but does not show the decimals
[FreqKhz] Only show the decimals
[FreqIndexN] Show only one number at index N of the frequency, where index starting from 3rd decimal to left (see example below)

Example 1:
[RadioName]
[FreqMhz].
[FreqKhz]
Output:
RADIO 1
144.
440

Example 2 (frequency is 144.440):
[RadioName]
[FreqIndex5][FreqIndex4].[FreqIndex3][FreqIndex2][FreqIndex1]
Output:
RADIO 2
44.440

Example 3:
[RadioName]
/[ [Clients] /]
Output:
RADIO 3
[ 9 ]

On transmission (if empty, it uses the default above):

Special sender name settings
Regex for sender name.
Example--- Remove characters within brackets [], <> and ():
\[.*?\]|\(.*?\)|\<.*?\>

Leave empty if you do not want to parse the sender name.

Set the maximum number of letters per line shown from the senders alias name.
I.e. if the name has 15 letters, and a value of 5 is set, the name will be divided in three lines of 5 letters each.
